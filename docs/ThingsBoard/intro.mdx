## ThingsBoard

ThingsBoard is an open-source IoT platform for data collection, processing, visualization, and device management. It provides a scalable and flexible solution for building IoT applications, enabling users to connect and manage their devices, collect and analyze data, and create custom dashboards and visualizations.

- https://thingsboard.io
- https://thingsboard.io/docs/

## Installation options

- ThingsBoard Cloud: A fully managed cloud service that allows you to quickly get started with ThingsBoard without worrying about infrastructure management.
- ThingsBoard PE (Professional Edition): A commercial version of ThingsBoard that offers additional features and
  support for enterprise use cases.
- ThingsBoard CE (Community Edition): An open-source version of ThingsBoard that can be self-hosted on your own infrastructure.
- ThingsBoard IoT Gateway: A lightweight gateway that can be deployed on edge devices to collect data from various sources and forward it to the ThingsBoard platform.
- ThingsBoard Docker: A containerized version of ThingsBoard that can be easily deployed using Docker.
