## OpenRemote

OpenRemote is an open-source IoT platform for data collection, processing, visualization, and device management. It provides a scalable and flexible solution for building IoT applications, enabling users to connect and manage their devices, collect and analyze data, and create custom dashboards and visualizations.

## Installation options

- OpenRemote Cloud: A fully managed cloud service that allows you to quickly get started with OpenRemote without worrying about infrastructure management.
- OpenRemote PE (Professional Edition): A commercial version of OpenRemote that offers additional features and support for enterprise use cases.
- OpenRemote CE (Community Edition): An open-source version of OpenRemote that can be self-hosted on your own infrastructure.
- OpenRemote IoT Gateway: A lightweight gateway that can be deployed on edge devices to collect data from various sources and forward it to the OpenRemote platform.
- OpenRemote Docker: A containerized version of OpenRemote that can be easily deployed using Docker.
