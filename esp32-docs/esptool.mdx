esptool is a popular, open-source Python utility used to flash, provision, and interact with Espressif Systems microcontrollers (like the ESP32 and ESP8266). It communicates directly with the chip's ROM bootloader to upload firmware binaries, read device information, and back up memory.

- https://docs.espressif.com/projects/esptool/en/latest/esp32/

Getting started is easy:

```bash
pip install esptool
```

You can check if it is working by running:

```bash
esptool chip-id
```

```bash
esptool v5.3.0
Connected to ESP32 on COM5:
Chip type:          ESP32-D0WD-V3 (revision v3.1)
Features:           Wi-Fi, BT, Dual Core + LP Core, 240MHz, Vref calibration in eFuse, Coding Scheme None
Crystal frequency:  40MHz
MAC:                f4:2d:c9:71:c6:e4

Stub flasher running.

Warning: ESP32 has no chip ID. Reading MAC address instead.
MAC:                f4:2d:c9:71:c6:e4

Hard resetting via RTS pin...
```

```bash
esptool.py --chip esp32 --port /dev/ttyUSB0 read_mac
```

Then you can use it to flash your ESP32:

```bash
esptool.py --chip esp32 --port /dev/ttyUSB0 write_flash -z 0x1000 esp32-20210902-v1.17.0.bin
```

```bash
esptool -h
```
