## Visão Geral sobre IoT

**Ementa:** O que é IoT · Referências · Seu uso · Internet e IoT

**Stack da semana:** ESP32 · PlatformIO · VS Code · Wokwi

| Aula | Tipo | Conteúdo                                         | Detalhamento                                                                                  |
| ---- | ---- | ------------------------------------------------ | --------------------------------------------------------------------------------------------- |
| 1    | AT   | Introdução à IoT: definição, história e evolução | Conceito de "coisa conectada", marcos históricos, impacto econômico e social                  |
| 2    | AT   | IoT industrial vs. IoT consumidor (IIoT)         | Diferenças de requisitos: confiabilidade, latência, segurança, escala                         |
| 3    | AP   | Setup do ambiente: VS Code + PlatformIO + Wokwi  | Instalação e configuração do ambiente de desenvolvimento; primeiro projeto ESP32 no simulador |
| 4    | AP   | Hello World IoT: piscar LED no ESP32 via Wokwi   | Estrutura de um projeto PlatformIO, `platformio.ini`, upload e monitoramento serial           |

---

- **Objetivo:** Configurar todo o ambiente de desenvolvimento.
- **Atividades:**
  1. Instalar VS Code + extensão **PlatformIO**.
  2. Criar projeto ESP32 no PlatformIO.
  3. Configurar e testar simulação no **Wokwi** (blink LED + integração com VS Code).
  4. Instalar via Docker: Node-RED, InfluxDB, Grafana e Mosquitto (MQTT broker).
- **Entrega:** Print do Wokwi rodando + containers Docker ativos (`docker ps`).
